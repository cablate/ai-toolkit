# Threads Post Optimizer

Systematic pre-publish checks for Threads posts, based on algorithm patent knowledge from `darkseoking-mindset`.

## How to Use

1. Receive draft → load [optimization-checklist.md](references/optimization-checklist.md) and run all checks
2. Rate each item **PASS / WARN / FAIL** with specific reasoning
3. If issues found → give concrete revision suggestions, re-check after edits
4. All PASS → ready to publish

## Scenes

| Scenario | Action |
|----------|--------|
| Have a draft to review | Run full checklist |
| Just went viral, planning next move | Run checklist #7 (Post-Viral Strategy) |
| Unsure if now is the right time to post | Run checklist #6 (Timing) |
| Don't know what to write | Run checklist #8 (Topic Suggestion) |
