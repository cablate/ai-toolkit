# Optimization Checklist

Rate each item PASS / WARN / FAIL.

---

## 1. First Line

Clickbait openers = low signal (US20190095961A1).

| FAIL | PASS |
|------|------|
| Sensational tone ("Shocking!", "X secrets nobody tells you", "99% get this wrong") | Hooks via specifics, counterintuitive claims, or real pain points |
| AI-typical openers ("In today's rapidly changing world", "Have you ever wondered") | High info density (numbers, proper nouns, scenarios) |
| Vague big words with no substance ("AI is amazing", "This tool changes everything") | darkseoking style: lead with conclusion or counterintuitive take |

---

## 2. Semantic Precision

Semantic vectors match first batch of users (US10579688B2). More precise = better match.

| FAIL | PASS |
|------|------|
| Dominated by generic terms ("AI tools", "digital transformation") | Domain-specific terms that make target audience think "this is for me" |
| Zero domain-specific nouns | Terminology appears naturally, not forced |

---

## 3. Topic Distance

Diversity mechanism (US9336553B2) penalizes high same-source density.

**Required info**: Topics of last 3-5 posts.

| FAIL | WARN | PASS |
|------|------|------|
| Same topic + same angle as previous post | Same broad topic, different angle (check interval is long enough) | Weakly related + different angle (same niche, different topic) |
| Completely unrelated to account positioning | | Fits Creator Embedding (US10558714B2) without repeating |

---

## 4. External Links

System evaluates domain authority + content relevance (EP2977948A1, US10268763B2).

| FAIL | PASS |
|------|------|
| Link unrelated to post content | No external links (safest) |
| Low-authority / unknown domain | Link has authority + highly relevant to content |

---

## 5. Thread Structure

Reference darkseoking viral data (darkseoking-all-posts.csv). All 4 posts above 1,000 likes had 6+ threads.

| FAIL | PASS |
|------|------|
| Main post can't stand alone — needs thread to make sense | Main post delivers value independently; threads deepen it |
| Thread replies repeat or paraphrase main post | Each thread reply adds independent information value |
| Content too thin to justify threads (padding) | Content depth warrants multi-thread structure (5-7 optimal) |
| Single post for complex topic that needs unpacking | Appropriate format for content density |

---

## 6. Timing

Rapid consecutive posting = diversity penalty + competing with yourself (US8402094B2, US10565267B1). Pausing actually enters priority distribution pool (US20140172877A1).

**Required info**: Last post time and performance.

| FAIL | PASS |
|------|------|
| Posting within 24h after a viral post | 24-48h+ since last post |
| Last post still in diffusion period | Last post performed average, no need to wait |

---

## 7. Post-Viral Strategy

**Only triggered when last post went viral.** Rate each sub-item.

| FAIL | PASS |
|------|------|
| Planning a follow-up post on the same topic and angle | Next post is weakly related, different angle (US9336553B2) |
| Rushing to post within 24h while viral post is still diffusing | Forced rest 24-48h, avoiding own diffusion cycle (US8402094B2) |
| Not engaging with comments on the viral post | Replying to every comment, liking everything, maximizing Expected Engagement Value (US9378529B2) |
| Posting without testing Audience Affinity first | Testing waters with comments on other posts before publishing |
| Chasing unrelated trending topics to ride momentum | Staying within Creator Embedding positioning (US10558714B2) |

---

## 8. Topic Suggestion

**Only triggered when user asks "what should I write?"**

| FAIL | PASS |
|------|------|
| Topic outside Creator Embedding positioning (infer from account history) | Topic aligns with proven sweet spot |
| Same topic + angle as one of the last 3-5 posts (diversity penalty risk) | Different angle or different topic within same niche |
| Second-hand commentary with no personal data | First-hand experience, mistakes, or results to share |
| Topic that only generates likes (low MSI weight) | Topic that sparks deep comments and DM shares (high MSI weight) |
